## Viabilidade confirmada em 16/03/2026

**Mercado:** 291 mil clínicas no Brasil, 81% MPEs, maturidade digital apenas 46%. TAM: R$ 1,2B/ano.

**Dores validadas:** No-show (20-30%, até R$ 144k/ano de prejuízo), agendamento manual (42% têm dificuldade), pós-atendimento inexistente.

**Canal:** WhatsApp domina (197M usuários BR, 90% dos médicos usam com pacientes).

**Concorrentes diretos:** Cloudia (líder, R$ 449-1.039/mês), Secretaria IA (R$ 597-997), WorkAI (R$ 5,5M funding), Botdesigner.

**Gap competitivo:** Mercado saturado em agendamento básico e chatbots de menu. Oportunidade em: IA conversacional real (LLM), pós-atendimento automatizado, cobrança integrada (PIX+boleto), setup instantâneo, preço acessível.

**Diferenciais SecretarIA:** Preço R$ 349 (22-42% menor), setup em minutos (vs 45 dias Cloudia), sem fidelidade, IA generativa real, cobrança Asaas integrada, Evolution API self-hosted.

**Why:** Pesquisa de viabilidade para iniciar o desenvolvimento do projeto.
**How to apply:** Usar estes dados para guiar decisões de produto, priorização de features e posicionamento de mercado.
